# EvaMassage - Chat Application

## Features
- User Registration & Login
- Real-time Messaging
- Search People
- Profile Management

## Deploy on Koyeb
1. Push this repository to GitHub
2. Connect to Koyeb
3. Set environment variables:
   - SECRET_KEY=your-secret-key
   - PORT=8080
4. Deploy!

## Local Development
```bash
pip install -r requirements.txt
python app.py
